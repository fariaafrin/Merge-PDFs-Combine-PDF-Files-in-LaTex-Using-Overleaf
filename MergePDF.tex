COMMAND
----------------------
Syntax:  \includepdf[key=val]{filename}
1. [pages={-}] => First to Last, All Pages
2.[pages={5-}] => 5th Page to Last Page
3. [pages={-5}] => 1st Page to 5th Page
4. [pages={2-5}] => Range: 2nd Page to 5th Page
5. [pages={3,6,9}] => Select only the 3rd, 6th, and 9th Pages
6. [pages={last}] => Select only the LAST Page
7. [pages={last-5}] =>Select All Pages in Reverse Order: From last to Fifth.
8. To INSERT EMPTY Pages Use -> {}
EXAMPLE: 
[pages={3,{},6-8,10}] -> will insert page 3, an empty page,and pages 6,7,8 and 10.
